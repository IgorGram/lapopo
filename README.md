# lapopo
Poker Planning

## Project setup
```
npm install
```

### Локальный запуск и горячая перезагрузка клиентской части проекта и API сервера
```
npm run serve
```
в другом терминале
```
npm run dev
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

### Heroku
Heroku по дефолту запускает по очередности 2 скрипта из файла package.json
```
npm run build
```
и
```
npm run start
```
